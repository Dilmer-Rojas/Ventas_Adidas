### Descripción breve del dataset:

Este dataset contiene información detallada sobre ventas de productos Adidas en Estados Unidos, incluyendo unidades vendidas, ingresos, ubicaciones, tipos de productos y otros datos relevantes para análisis.

### Objetivos del análisis:

- Análisis de productos más vendidos: identificar los productos más vendidos y analizar sus características (precio, categoria, etc).

- Análisis de regiones y ciudades con mayor demanda: Analizar las ventas por región y ciudad para identificar áreas con mayor demanda.
